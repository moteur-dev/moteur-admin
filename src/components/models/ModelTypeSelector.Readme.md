# ModelTypeSelector

Visually select the type of model you're creating.

## Options

- **Content** – Regular content (posts, pages, blocks)
- **User Data** – Information submitted by users (profiles, forms)
- **Taxonomy** – Categories, tags, filters
- **Settings** – Configuration data or singletons

## Usage

```tsx
<Form.Item name="type" label="Model Type">
  <ModelTypeSelector />
</Form.Item>
```
